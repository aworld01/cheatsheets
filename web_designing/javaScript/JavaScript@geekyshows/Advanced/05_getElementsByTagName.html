getElementsByTagName("Tag_Name")

The method getElementsByTagName("Tag_Name") returns a live node list meaning that it updates itself with the DOM tree automaticallly, so modification of the DOM tree will be reflected in the returned collection. The returned Node List or Collection of Nodes can be accessed by index numbers starting with index 0.

  • This method accepts a string indication the type of elements that be retrieved, a special value "*" returns all elements in the documents.
  • You can use the length property of the NodeList object to determine the number of elements with the specified tag name, then you can loop through all elements and exatract the info you want.

getElementsByTagName("p")

Where p is name of Tag.

getElementsByTagName("*")

The parameter value "*" returns all elements in the document.
