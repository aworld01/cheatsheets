transition

ease: It specifies a transition effect with slow start, then fast, then end slowly (default) (equivalent to cubic-bezier(0.25,0.1,0.25,1)) linear: It specifies a transition effect with the same speed from start to end. (equivalent to cubic-bezier(0.25,0.1,0.25,1)) ease-in: It specifies a transition effect with a slow start (equivalent to cubic-bezier(0.42,0,1,1)) ease-out: It specifies a transition effect with a slow end (equivalent to cubic-bezier(0,0,0.58,1)) cubic-bezier(.3,.7,.95,.39): Define your own values in the cubic-bezier function. Possible values are numeric values from 0 to 1. ease-in-out: step-start step-end steps(int, start or end)
