box-shadow

syntax:- box-shadow: h-shadow v-shadow blur spread color inset; box-shadow: 30px 30px; box-shadow: 30px 30px 15px; box-shadow: 30px 30px 15px 20px; box-shadow: 5px 5px 10px 3px red; /* we can add more then one shadow */ box-shadow: 5px 5px 10px 3px red, 10px 10px 20px 6px green; h-shadow: The position of the horizontal shadow. Nagative values are allowed. v-shadow: The position of the vertical shadow. Nagative values are allowed. blur: The blur distance. spread: Size of shadow. Nagative values are allowed. color: The color of shadow. inset: Changes the shadow from an outer shadow (outset) to an inner shadow.
