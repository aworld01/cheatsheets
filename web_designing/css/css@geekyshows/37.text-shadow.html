text-shadow

It specifies the shadow effect added to text. We can add more than one shadow to the text, separated by comma.

  1. h-shadow - The position of the horizontal shadow. Negative values are allowed.
  2. v-shadow - The position of the vertical shadow. Negativ e values are allowed.
  3. blur-radius - The blur radius. Default value is 0.
  4. color - The color of the shadow.
  5. none - No shadow.

text-shadow: 2px 2px 5px red;

text-shadow: 2px 2px 5px red, 5px 5px 8px blue;
