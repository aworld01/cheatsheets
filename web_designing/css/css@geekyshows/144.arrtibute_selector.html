Attribute selector

We can set style using attribute or its value (class or id)......

p[class]{color: green;} Set style using attribute and its value.

p[class="game"]{color: green;} Set style using attribute and its value.

[class~="game"]{color: green;} Set style using an attribute value containing a specified word.

[class|="game"]{color: green;} Set style using with the specified attribute starting with the specified value. The value must be a whole word.

[class^="game"]{color: green;} Set style whose attribute value begins with a specified value.

[class$="game"]{color: green;} Set style whose attribute value ends with a specified value. It's not necessary that the value should be a whole world.

[class*="ga"]{color: green;} Set style whose attribute value contains a specified value.
