align-items

This property is used to specify the defalut alignment for items inside the container. This property can be override using align-self property for each item. We can set this property to stretch(default), center, flex-start, flex-end, baseline. It's applys on with height.

align-items: stretch; (default)
align-items: flex-start;
align-items: flex-end;
align-items: center;
align-items: baseline;

align-items: stretch; (default): This is used to stretch the item to fit the container. It won't work if you spacify height already.

align-items: flex-start; This is used to position the item at the beginning of the container.

align-items: flex-end; This is used to position the item at the end of the container.

align-items: center; This is used to position the item at the center of the container.

align-items: baseline; This is used to position the item at the baseline of the container.
