Media Query

A media query consists of an optional media type and zero or more expressions that the limit the style sheets' scope by using media features, such as width, height, and color.

@media screen and (max-width: 1200px) {.lrm{ background-color: mediumorchid; }} @media screen and (max-width: 900px) {.lrm{ background-color: red; }}
