float and clear

    float

  1. float: none; (defalut)
  2. img{float: right;}
  3. img{float: left;}

    4. clear

  4. img{float: right;} p{clear: right;}
  5. img{float: left;} p{clear: left;}
  6. img.left{float: left;} img.right{float: right;} p{clear: both;}

float: This property is used to specify that an element should be placed along the left or right side of its container, where text and inline element will wrap around it. We can set this property to left, right, and none (default).

clear: This property is used to specify that an element should be placed along the left or right side of its container, where text and inline element will wrap around it. We can set this property to left, right, and none (default).
