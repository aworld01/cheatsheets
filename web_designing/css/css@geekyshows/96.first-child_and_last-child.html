Pseudo class (:first-child, and :last-child)

first-child will set CSS rule to every first element of selector which is someone's child.

last-child will set CSS rule to every last element of selector which is someone's child.

:first-child :last-child :first-of-type :last-of-type :nth-child(n) :link :visited :hover :active :focus :checked
