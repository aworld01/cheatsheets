first-letter and Last-letter

It's used to add style to the first-letter of the specified selector(each). This can only be used with block-level elements. - background properties - font properties - padding properties - color properties - margin properties - border properties - line-height - text-transform - text-decoration - vertical-align (only if float is 'none') - float - clear
