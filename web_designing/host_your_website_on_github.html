Host a static website on Github

Requirements

	01: A ID on github.com
	02: Code for a static website
	03: Make sure in the root directory of your code you have 'index.html'
	04: Goto Github.com and Sign In
	05: Create a public repository (the repository name should be ends with '.io')

Open gitbash and fire these commands:-

	git init
	git status
	git add .
	git status
	git commit -m "initial_commit"
	git status
	git remote add origin "url of your repository"
	git push origin master

Goto=> settings/pages/

Source [master]
Save

Your site is ready to be published at https://aworld01.github.io/aw01/
Note: Wait for 4 to 5 hours and your website can be access now.
