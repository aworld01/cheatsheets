HTML Tags

  1. title tag: syntax:- <title>This is title tag...</title>
  2. body bgcolor tag: syntax:- <body bgcolor="skyblue"></body>
  3. body background tag: syntax:- <body background="Images/italy.jpg"></body>
  4. i tag: syntax:- This is <i>italic</i> tag...
  5. u tag: syntax:- This is <u>underline</u> tag...
  6. b tag: syntax:- This is <b>bold</b> tag...
  7. s tag: syntax:- This is <s>strike/del</s> tag...
  8. sup tag: syntax:- This is <sup>super</sup>script tag...
  9. sub tag: syntax:- This is <sub>sub</sub>script tag...
  10. font color tag: syntax:- This is <font color="red" size="2" face="arial">font tag...</font>
  11. h1 tag: syntax:- <h1>This is heading tag...</h1> (h1-h6)
  12. <!-- This is a HTML Comment -->
  13. img src tag: syntax:- <img src="Images/italy.jpg" alt="Image" title="This is an Image." width="300" height="200">
  14. br tag: syntax:- This is<br />my friend.
  15. hr tag: syntax:- He is<hr />your friend.
  16. Who are &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; you?
  17. center tag: syntax:- <center> <img src="Images/italy.jpg" alt="Image" title="This is an Image." width="300" height="200"> </center>
  18. marquee tag: syntax:- <marquee>This is marquee tag example.</marquee>
  19. a href tag: syntax:- <a href="google.com">Google</a>
