Python3 commands

  1. python --version: to check python version.
  2. python (python3 if you are using Linux OS): to lunch python interpreter.
  3. exit(): to exit from python interpreter.
  4. python hello.py (python3 hello.py if you are using Linux OS): to execute python file.
  5. pip --version: to check pip version.
  6. pip install --upgrade pip: to update pip
  7. pip install pyttsx3: to install python package.
  8. pip install pyttsx3==2.87: to install a specific version of any package.
  9. pip uninstall pyttsx3: to uninstall python package.
  10. pip list: to check installed python packages and version.
  11. pip help: to help.
