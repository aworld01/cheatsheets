Shortcut key

Ctrl+B
Hide/Unhide Explorer

Selection

Alt+Click
to insert multipleCursors.
Ctrl+Alt+Up/Down
Insert cursors above or below.
Shift+Alt+I
Insert cursors at the end of selected lines.
selectSomeText, Ctrl+D
Insert a cursor at the next matching location.
selectSomeText, Ctrl+Shift+L
Insert a cursor at all matching location
CTRL + F2
to change all same words then press ESC.
CTRL + G "line number"
to goto any line by line no.
CTRL + SHIFT + T: to reopen file.
  • ALT + Down auro: to move Line down.
  • ALT + Up auro: to move Line up.
  • SHIFT + ALT + Down auro: to copy line down.
  • SHIFT + ALT + Up auro: to copy line up.
  • CTRL + /: to comment/uncomment.

    • Emmet

  • p: to paragraph.
  • b: to bold tag.
  • i: to italic.
  • br: to br tag.
  • div: to make div.
  • h1: to heading 1 tag.
  • h.classname: to make a class in h1 tag.
  • h1.classone.classtwo: to make two classes in h1 tag.
  • h1#idname: to make a id in h1 tag.
  • .classname: to make a class in div.
  • h1#myId.myClass: to make one id and one class in h1 tag.
  • h1#myId.myClassOne.myClassTwo: to make one class and two id in h1 tag.
  • ftr: to make footer tag.
  • hdr: to make header tag.
  • p{This is paragraph}: to type "This is paragraph" in p tag.
  • h1*4: to type h1 four times.
  • p.myClass{This is paragraph}: to type "This is paragraph" in p tag with class.
  • h1>p: to make child paragraph in h1 tag.
  • h1>ul>li: to make grand-child list in h1 tag.
  • h1>ul>li{This is list one}: to type "This is list one" in grand-child list in h1 tag.
  • h1>ul>li{This is list}*4: to make 4 grand-child list and type "This is list" in h1 tag.
  • h1>ul>li{This is list $}*4: to make 4 grand-child list and type "This is list" dynamicaly in h1 tag.
  • h1{This is heading one}>h2{This is heading two}>p{This is paragraph.}: to type sentences with individual child tag.

    h1{This is heading one}+h2{This is heading two}+p{This is paragraph.}
    to type sentences with individual siblings tag.
    h1.myClass+h2.myClass2+p
    to type classes with individual siblings.
    link:css
    to link css file
    script:src
    to link script file

  • form: to make a form.
  • form:post: to make a form with post method.
  • inp: to input tag.
  • input:password: to make input with password type.
  • lorem: to type dummy text.
  • lorem7: to type dummy text in seven word.
  • h1>p.myClass>lorem: to type dummy text in child paragraph with a class.
  • div>ul>lorem5.item*3: to type three times dummy text in list limit with 5 words.
