Installation NetHunter

  1. Install the NetHunter-Store app from https://store.nethunter.com
  2. From the NetHunter Store: install Termux, NetHunter-KeX client, and Hacker's keyboard
    3. Note: The button Install may not change to Installed in the store client after installation - just ignore it.
  3. Starting termux for the first time may seem stuck while displaying installing on some devices - just hit Enter.
Usage:

Open Termux and type one of the following::-

  1. nethunter: to start Kali-NetHunter command line interface
  2. nethunter kex passwd: to configure the KeX password (only needed before 1st use)
  3. nethunter kex &: to start Kali NetHunter Desktop Experience user sessions
  4. nethunter kex stop: to stop Kali NetHunter Desktop Experience
  5. nethunter pwd: to run pwd command in NetHunter environment
  6. nethunter -r: to start Kali-NetHunter CLI as root
  7. nethunter -r kex passwd: to configure the KeX password for root
  8. nethunter -r kex &: to start Kali NetHunter Desktop Experience as root
  9. nethunter -r kex stop: to stop Kali NetHunter Desktop Experience root sessions
  10. nethunter -r kex kill: to kill all KeX sessions
  11. nethunter -r apt update: to run update command in NetHunter environment as root
  12. nethunter -r kex stop: to stop Kali NetHunter Desktop Experience root sessions
