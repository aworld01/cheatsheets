Git 2.29.2

KaliLinux 2020.1(cd /home/kali/Documents)

Goto your folder path, open Terminal and fire the commands below:-

cd /c
to change the drive.
git config user.name
to check entered name.
git config user.email
to check entered email.
git config --global core.editor nano
to set default editor.
git --version
to check the version of git.
git update-git-for-windows -y
to update git.
mkdir 'myGit'
to make a folder name 'myGit'.
ls -la
to list and check permissions of all files and folders including hidden files and folders also.
rm -rf .git
to delete git folder's repository.
CTRL+L
to clear command line.
git add 'filename/foldername'
to stage a specific file/folder.
git add .
to stage all file/folder.
git log
to check log of commits.
git show (commit id)
to check commited files information.
git log
to check log of commits.
git log -1
to check last update.
git log -2
to check last two updates.
git log --oneline
to see all commits in oneline.
git log --grep "initial_commit"
to check any commit by commit message.

Cloning

Goto 'duckduckgo.com' and type 'github website tensorflow' and click on the first link. Click on Clone and copy the URL. Now open Git Bash and type:- git clone 'put cloning url" myFolder: to clone git repository into the 'myFolder' directory. git clone 'paste your copied url': to clone git repository in the default directory. cd tensorflow: to goto tensorflow folder. ls: to check list of files and folders. git log: to check log of commits. Press 'q' to exit.
