Git 2.29.2

KaliLinux 2020.1(cd /home/kali/Documents)

Goto your folder path, open Terminal and fire the commands below:-

git commit -am "direct commit"
to direct commit modified files/folders. (skip staging area)
git rm 'filename'
to remove and stage the file.
git mv 'filename' 'new_filename'
to rename and stage the file.
git restore --staged .
to unstage all files.
git checkout -- myFile.txt
to unmodify myFile.txt
git checkout -f
to unmodify all modified files and goto your last commit.

Check log

git log
to check log. press "q" to quit.
git log -p
to check diffrences of log commits.
git log -p -5
to check diffrences of only 5 log commits.
git log --stat
to check diffrences in short list.
git log --pretty=short
to see all commits in short list
git log --pretty=full
to get more commits information.
git log --pretty=oneline
to see all commits in only one line.
git log --since=2.days
to see last two day's log.
git log --since=2.weeks
to see last two week's log.
git log --since=2.months
to see last two month's log.

To check difference of modified files.


git diff
to check last modified files diffrences.
git diff --staged file2.txt
to compare difference between last commited file2.txt to current staging area file2.txt.

Branching(to save repository from any changes)

git branch
to check branch list.
git branch "new branch name"
to make a new branck.
git checkout "branch name"
to switch branck.
git branch

git checkout -b "new branch name"
to make a new branck and switch on.
git branch
to check branch list and selected branch.
Goto git folder and modify some file/folder.
git status
to check status.
git add .
to stage all files and folders.
git commit -m "made a new branch and modified some file/folder."
to create a checkpoint.
git checkout 'old branch name'
to switch to the old branch.
Goto git folder and check, all "old branch's" files and folders will be there as it was as.
git merge myBranch
to merge branches.
git push origin master
to push repository to remote.
git push origin "local branch" : "remote branch"
to push local branch to remote with a new name.
Goto: github.com reload your page and see changes.
git branch -d myBranch
to delete local branch.
git branch -v
to see branch list with last commit.
git branch --merged
to see all merged branch.
git --no-merged
to see not merged branch.
git remote
to see remote branches.
git remote -v
to see remoted website.
git push -d origin "branch name"
to delete remote branch.
git push origin --delete myBranch
to delete remote branch.
