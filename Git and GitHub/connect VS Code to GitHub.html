Connect Visual Studio Code to GitHub

  • Download and install Git:
  • Download and install Visual Studio Code:
  • Make an account at github.com
  1. Open Visual Studio Code and select the folder that you want to backup.
  2. Goto Source Control (CTRL + SHIFT + G) click on intitialize Repository.
  3. Type your commit message in the Message box and click on Commit/Always.
    4. (Note: configure email and name if require)
  4. Go and login to your github account.
  5. Click on right side git icon/Your repositories/New and make a repository.

    Repository name: my_backup_files
    Description (optional): Making this repository only for backup all my files.
    Create repository

  6. Copy the line that will be look like git remote add origin git@github.com:artificialworld09/my_backup_files.git"
  7. Goto Visual Studio Code/View/Terminal and paste the line that you just copied, hit ENTER
  8. Goto Source Control (CTRL + SHIFT + G) click on Three dots/Push/OK
    9. (Note: give email id and password if require)
  9. Goto github, reload your page and see your repository has been uploaded there.
  10. Now goto Visual Studio Code and modify some files/folders then goto Source Control (CTRL + SHIFT + G) and type your commit message in the Message box and click on Commit
  11. Then click on Three dots/Push/OK
