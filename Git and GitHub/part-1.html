Make a '.gitignore' file to ignore commit

Open Git Bash and type commands below:-

nano .gitignore file1.txt *.html myFolder CTRL + S, CTRL + X
nano .gitignore #to make a ignore file.
file1.txt #to ignore any specific file like (file1.txt).
*.html #to ingore all files with any specific extension like (.html).
myFolder #to ignore all files in any specific folder like (myFolder).
CTRL + S: to save.
CTRL + X: to exit.
cat .gitignore: to check git file that you made just.

git status

git add .

git commit -m 'added gitignore'

git log
to check log of commits. Press 'q' to exit.
