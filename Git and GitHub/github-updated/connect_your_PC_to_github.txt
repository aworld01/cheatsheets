1. Install Git
If Git isn’t installed yet:
    Download from git-scm.com
    Install with default settings

2. Set Up Git with Your GitHub Account
Open Command Prompt or Git Bash and run:
    git config --global user.name "Your GitHub Username"
    git config --global user.email "your-email@example.com
    git config --list
	

3. Generate SSH Key (Recommended for Secure Connection):
    ssh-keygen -t ed25519 -C "your-email@example.com"

Press Enter to accept default file location

Set a passphrase if you want (optional):
Example of Good passphrase:
        "MyScooterTailLampFix2023WasTricky!"

Then run:
    cat ~/.ssh/id_ed25519.pub

Copy the output (your public key).

4. Add SSH Key to GitHub
Go to GitHub → Settings → SSH and GPG keys:
    Click “New SSH key”
    
    Title: TinyCore
    Key:
	Paste the copied key and save

5. Test the Connection:
    ssh -T git@github.com

You should see a message like:
Hi Abdul! You've successfully authenticated...

6. Clone a Repository or Create One
To clone:
    git clone git@github.com:username/repo-name.git


To create a new repo:
    On GitHub, click “New repository”

Then run:
    git init
    git remote add origin git@github.com:username/repo-name.git


Use a Passphrase with ssh-agent
===============================
1. Create Your SSH Key with a Passphrase if you haven't already:
    ssh-keygen -t ed25519 -C "your-email@example.com"

When prompted:
    File location: Press Enter to accept default

    Type a strong, memorable passphrase
    Confirm: Retype it

2. Start the SSH Agent
This tool stores your key in memory so you don’t have to retype the passphrase:
    eval "$(ssh-agent -s)"

3. Add Your Key to the Agent:
    ssh-add ~/.ssh/id_ed25519

You’ll be asked for your passphrase once.
After that, the agent remembers it until you log out or restart.

4. Optional: Auto-Start Agent on Login (Windows)
If you're using Git Bash or WSL - Add this to your '~/.bash_profile' or '~/.bashrc':
    eval "$(ssh-agent -s)"
    ssh-add ~/.ssh/id_ed25519
