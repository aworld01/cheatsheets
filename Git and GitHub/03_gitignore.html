.gitignore

Ignore unwanted Files & Folders:-

nano .gitignore 
    file1.txt
    myFolder/
    *.mp3
    *.gif
    *.mp4
Ctrl+S, Ctrl+X
cat .gitignore
to check ".gitignore" file's data
git status
to check status
git add .
to create checkpoint
git commit -m "added: '.gitignore' file"
to create checkpoint
git rm --cached *.mp3
to remove file cached history
