System Administrative

Run commands

cleanmgr
to run diskCleanupManager
chkdsk /r X:
to repair corrupted memory (X: is memory drive)
cmd, Ctrl+Shift+Enter, Alt+Y
to open cmd as administrator

CMD commands

Get PC details

wmic computersystem get model
to get model number
wmic computersystem get name
to get computerName
wmic computersystem get manufacturer
to get manufacturer
wmic computersystem get systemtype
to get systemType
wmic computersystem get manufacturer, model
combine commands
hostname
to get hostName
wmic computersystem where caption = "Desktop1-PC" rename "usr1-PC"
to change hostName
