Make multiple partition in usb drive

Open CMD and type the following commands

diskpart: to open partition manager. list disk: to check the disk in list. select disk 1: in my case 'disk 1' is usb drive, you will select your disk by list above. clean: to clean disk configration. create part pri size=92160: to create partition. ('size=92160' is the size of disk) format fs=ntfs label="softwares" quick: to format and create name of partition. assign letter=x: to assign the partition letter. create part pri: to create partition with balace space. format fs=ntfs label="my_files" quick assign letter=z
