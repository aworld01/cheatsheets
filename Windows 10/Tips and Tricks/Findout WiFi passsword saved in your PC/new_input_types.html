Find your Wi-Fi network password in Windows 10

If you forgot your Wi-Fi network password, you can find it if you have another Windows PC already connected to your Wi-Fi network. After you find your password, you can use it on another PC or device to connect to your Wi-Fi network. select the Start button, then select Settings > Network & Internet > Status > Network and Sharing Center. In Network and Sharing Center, next to Connections, select your Wi-Fi network name. In Wi-Fi Status, select Wireless Properties. In Wireless Network Properties, select the Security tab, then select the Show characters check box. Your Wi-Fi network password is displayed in the Network security key box.
