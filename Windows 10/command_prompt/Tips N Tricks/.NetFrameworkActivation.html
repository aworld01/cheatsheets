CLI (Command Line Interface)
Date: 02/05/2019

Activate
.Net Framework
service in Windows-8

  1. Enter your Windows-8 Installation Disk on DVD Drive.
  2. Run Command Prompt as Administrator:-
    1. Press
      WIN
      +
      X
    2. Press
      A
    3. Select
      Yes
  3. Now copy the following command and paste it into the Command Prompt window (To paste the command in Command Prompt window, click the right button of mouse and select paste).
    Or type the following line in command prompt and press
    ENTER
    4. DISM /Online /Enable-Feature /FeatureName:NetFx3 /All /LimitAccess /Source:D:\sources\sxs
    ENTER

    Remember:
    D
    is the drive letter of DVD drive. Change it with your DVD drive letter.
  4. Now
    .Net Framework
    will be installed within few minutes.
