CLI (Command Line Interface)
Date: 19/05/2019

CMD
net user
commands

  1. C:\Windows\system32>
    net user
    ENTER

      This command is used to to see how many UserAccounts are available in your PC.
  2. C:\Windows\system32>
    net user "abdul zoha"
    ENTER

      This command is used to see any UserAccount status.
  3. C:\Windows\system32>
    net user "abdul zoha" /add
    ENTER

      This command is used to add a new LocalUserAccount in your PC.
  4. C:\Windows\system32>
    net localgroup administrators "abdul zoha" /add
    ENTER

      This command is used to convert any LocalUserAccount into AdministratorAccount.
  5. C:\Windows\system32>
    net user "abdul zoha" /active:yes
    ENTER

      This command is used to activate the User.
  6. C:\Windows\system32>
    net user "abdul zoha" /active:no
    ENTER

      This command is used to deactivate the User.
  7. C:\Windows\system32>
    net user "abdul zoha" /delete
    ENTER

      This command is used to delete the User.
  8. C:\Windows\system32>
    net user "abdul zoha" 12345
    ENTER

      This command is used to change the User password (direct).
  9. C:\Windows\system32>
    net user "abdul zoha" *
    ENTER

      This command is used to change/delete the User password (optional).
