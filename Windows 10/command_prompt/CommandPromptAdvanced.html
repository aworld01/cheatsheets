  1. C:\Windows\system32>
    systeminfo
    ENTER

      This command is used to get all information about your Computer.
  2. C:\Windows\system32>
    ver
    ENTER

      This command is used to get information about our OS version.
  3. C:\Windows\system32>
    time
    ENTER

      This command is used to change the Time of PC.
  4. C:\Windows\system32>
    date
    ENTER

      This command is used to change the Date of PC.
  5. C:\Windows\system32>
    vol
    ENTER

      This command is used to get information about our working Drive/Volume.
  6. C:\Windows\system32>
    wmic logicaldisk get name
    ENTER

      This command is used to see that how many Drives/Volumes are there available in your computer.
  7. C:\Windows\system32>
    d:
    ENTER

      This command is used to switch to the other Drive of Computer", like "C:\" "D:\" "E:\" etc.
  8. C:\Windows\system32>
    dir /a /p
    ENTER

      This command is used to see the all hidden file and directory page wise.
  9. C:\Windows\system32>
    copy *.mp3 "f:\music"
    ENTER

      This command is used to copy all ".mp3" extention's file from current working directory to "F:\music" drive.
  10. C:\Windows\system32>
    ren *.txt *.dat
    ENTER

      This command is used to rename ".txt" file into ".dat" file.
  11. C:\Windows\system32>
    ren *.* *.dat
    ENTER

      This command is used to change all files extention of current working directory into ".dat" file.
  12. C:\Windows\system32>
    del *.txt
    ENTER

      This command is used to delete all ".txt" extention's file.
  13. C:\Windows\system32>
    del /f /s /q /a windows.old
    ENTER

      This command is used to delete all files in "windows.old" folder but not it's subfolders.
  14. C:\Windows\system32>
    del /f /s /q /a *
    ENTER

      This command is used to delete all files of current working folder but not it's subfolders.
  15. C:\Windows\system32>
    rd /s /q windows.old
    ENTER

      This command is used to delete "Windows.old" folder & all it's content without any notification.
  16. C:\Windows\system32>
    rd /s /q *
    ENTER

      This command is used to delete current working folder & all it's content without any notification.
  17. C:\Windows\system32>
    xcopy /s /q /f /a c:\windows f:\
    ENTER

      This command is used to copy the File, Folder and Subfoler without asking any confermation.
  18. C:\Windows\system32>
    taskmgr
    ENTER

      This command is used to open the "Task manager".
