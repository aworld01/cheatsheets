Kali Lunux 2020.1

Nano 4.8 (text editor)

  1. man nano: to get manual about nano.
  2. /your word: to find anything in manual.
  3. q: to quit manual
  4. nano: to open nano editor.
  5. CTRl + G / F1: to display help menu.
  6. CTRL + E: to goto the end of the line.
  7. CTRL + A: to goto the begining of the line.
  8. CTRL + Y: page down.
  9. CTRL + V: page up.
  10. ALT + /: to goto the buttom of the page.
  11. ALT + \: to goto the start of the page.
  12. CTRL + X / F2: to close the current buffer / Exit from nano.
  13. CTRL + S: to save the file.
  14. CTRL + O: to save as the file.
  15. CTRL + K :to to cut the current line.
  16. CTRL + R "file_name": to add another text file data to current file.
  17. ALT + A (Select text by auro key) then ALT + ^ to copy the text.
  18. CTRL + U: to paste.
  19. CTRL + W "your word": to find any word in file.
  20. ALT + R "your word" "replace word: to replace any word.
  21. ALT + G "line number": to goto any line by line no.
  22. CTRL + R: to open file.
