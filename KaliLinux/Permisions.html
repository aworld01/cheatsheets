File permisions = chmod

3 types fact

To check the permisions of any type of file: ls -l

Indicator:Permissions:User:Group:Filesize:Date:Time:Filename

Indicator

~: RegularFile
d: Directory
l: Symblic link

Type of permissions:-

1: User permission
2: Group permission
3: Other user permission
u: User
g: Group
o: Other
a: All
-R: Recursive
+: Add permission
-: Remove permission
=: Replace permission
4 = read
2 = write
1 = execute

02: Binary reference/numeric permissions

Change the permision of any type of file by binary reference: chmod 777 -R 'myFolder'

01: Explicitly/symbolic permissions

Admin: 1 + 2 + 4 = 7 Read Write Execute
Group: 4 + 2 = 6 Read Write
Other: 4 Read

Change the permision of any type of file by symbolic type: chmod a+rwx -R 'myFolder'

Now the complete permision looks like: ~ rwxrwxrwx

Change owner = chown

chown userName File/DirectoryName: to change owner only.

chown userName:groupName File/DirectoryName: to change owner and group at the same time.

chown :groupName File/DirectoryName: to change group only.

chown user:group -R myFolder: to recursively change user and group of 'myFolder'.
