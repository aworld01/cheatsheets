Make Global alias in linux

Method-1

Goto: /home/'user'/, edit .bashrc file and add the line below:- alias cls=clear: to clear the 'Terminal'. alias ll='ls -l': to see the long list. Now save the file, open Terminal and fire source .bashrc: command to apply changes.

Mehod-2

sudo nano .bashrc # Alias definitions. # You may want to put all your additions into a separate file like # ~/.bash_aliases, instead of adding them here directly. # See /usr/share/doc/bash-doc/examples in the bash-doc package. if [ -f ~/.bash_aliases ]; then . ~/.bash_aliases fi CTRL+S,CTRL+X

sudo nano .bash_aliases alias 'kali --version'='lsb_release -a' alias desk='cd /home/thor/Desktop' alias doc='cd /home/thor/Documents' alias downl='cd /home/thor/Downloads' alias ll='ls -l' alias la='ls -la' alias de='rm -rf' alias cls='clear' alias sedit='sudo nano'
