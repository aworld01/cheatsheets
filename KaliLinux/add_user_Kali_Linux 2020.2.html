Kali Linux (user-account)

Open Command prompt and fire these commands to add a user in Kali Linux:-

  1. useradd -m thor# to add a user named thor.
    (use tail -5 /etc/passwd# to check added or deleted user.)

  2. passwd thor# to create a password for thor user.
    New password: *****
    Retype new password: *****
    passwd: password updated successfully

  3. usermod -a -G sudo thor# to Add the user to the sudo group (to install software etc).

  4. chsh -s /bin/bash thor# Now finally change the default shell of the newly added user to bash.
    ("-s" to provide a new login shell for the user account)

  5. su - thor# to switch the user.

  6. sudo su# to first time login to the root user when you install kali.

  7. su -# to switch to the root user.

  8. useradd -d /temp/ thor# to add user with any specific location home directory.

  9. useradd -u 1010 thor# to create a user with a specific id.

  10. usermod -d /home thor# to change user home directory.

  11. usermod -c "This is thor shell" thor# to give full name or write comment.

  12. sudo userdel -r thor# to completely delete user and his directories.

  13. chage thor# to set user expiry date etc.

  14. groups# to check the groups of current user.

  15. id thor# to check user group and id.

  16. passwd -l thor# to lock password.
    (use cat /etc/shadow# to check password status.)

  17. passwd -u thor# to unlock password.
    (use cat /etc/shadow# to check password status.)

  18. usermod -l 'newName' 'oldName'# to change the login name of user.

  19. usermod -u 1011 thor# to change the user id.

  20. usermod -G superman thor# to be the member of any group.

  21. usermod -G superman batman thor# to be the member of multiple group.

  22. usermod -a -G root thor# to change the user group.

  23. usermod -g superman thor# to change "primary group" of any user.

    24. Kali Linux (user-group)

    There are two types of group:-

    01: Primary group (it group automatically created when we add a user.
    02: Supplymentry group (we have to create this group)


  24. groupadd superman# to add a group named superman.
    (use tail -5 /etc/group# command to check added or deleted group.)

  25. groupadd -g 2020 superman# to create a group with a specific id.

  26. groupmod -g 1006 superman# to change group id.

  27. groupmod -n 'newName' 'oldName'# to change the group name.

  28. gpasswd -M user1,user2 'groupName'# to add multiple users into a group.

  29. newgrp 'groupName'# to login group.

  30. gpasswd 'groupName'# to set a group password.
    (use cat /etc/gshadow# command to check password is applied or not.)

  31. groupdel superman# to delete group.

When a new user is created. These files are modified.

  1. /etc/passwd
  2. /etc/shadow
  3. /etc/group
  4. /etc/gshadow

A new directory is created under /home/ with the name of the user.
It will automatically have all the contents from /etc/skel/ directory copied into it.
