Clone whole Website

Open Terminal and type:

mkdir clonedWebsites: to make a 'clonedWebsites' named folder.
cd clonedWebsites: to go into 'clonedWebsites' named folder.
wget -mk https://www.w3schools.com: to clone the whole site.
