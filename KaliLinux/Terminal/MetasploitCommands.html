MetaSploit v5.0.89-dev

  • msf5 > msfconsole
    to lunch Metasploit.
  • msf5 > banner
    to change MetaSploit banner.
  • msf5 > search windows
    to search payload and Sploits for Windows OS.
  • msf5 > info "Sploit name"
    to get information about any sploit
  • msf5 > clear
    to clear the MetaSploit Terminal.
  • msf5 > CTRL + C + ENTER
    to stop any running command.
  • msf5 > exit
    to exit from MetaSploit.
