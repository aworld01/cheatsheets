KaliLinux_2020.2

Press: CTRL + ALT + T to open terminal.

  • lsb_release -a
    to check the version of KaliLinux
  • sudo apt-get update
    to update Linux OS.
  • sudo apt-get upgrade
    to upgrade Linux OS.
  • sudo apt-get dist-upgrade
    to upgrade core Linux OS.

    • Or

  • sudo apt-get update && apt -y full-upgrade
    to full upgrade your Linux OS.
  • sudo chsh -s /bin/zsh
    to change the user shell.
  • sudo apt-get install -y virtualbox-guest-x11
    to install guest edition in your virtual box.
  • sudo apt-get install vlc
    to install vlc player
  • sudo apt-get remove -y vlc
    to remove vlc player
  • sudo apt-get --purge remove vlc
    to remove configration file of any software after removing that.
  • pwd
    to check persent working directory.
  • mkdir 'new_folder'
    to make a 'new_folder'.
  • ls -la
    to check all list of files and directory.
  • rm -rf 'New folder'
    to forcely delete any file and folder.
  • whatis rm
    to get a short details about rm.
  • man rm
    to get a manual about rm.
  • man ls > command_menu.txt
    to print ls command manual into a plain text file.
  • rm --help
    to know uses and santax of rm command.
  • which rm
    to know the path of rm exist.
  • which python3
    to know the path of python3 exist.
  • locate -i leafpad
    to find any file.
  • bg
    to see stopped program
  • fg 1
    to run background stopped program
  • jobs
    to see background program status.
  • ps a
    to see the list of running tasks.
  • kill 729
    to kill any running task. (729 is PID)
  • kill -9 729
    to forcely kill any running task.
  • ps aux
    to see all running procese.
  • ps aux | less
    to see all running procese line by line.
  • CTRL + C
    to kill any running command.
  • CTRL + Z
    to stop any running command.
  • reboot
    to restart the operating system.
  • shutdown now
    to shutdown the operating system.
