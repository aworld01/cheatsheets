Install Virtualbox

Goto the Terminal window and type:

sudo apt-get install build-essential dkms
sudo apt-get install virtualbox

Now type virtualbox and have fun...

Install OpenOffice

Goto the file location, open Terminal window and type:

tar xvf 'filename' then hit ENTER

cd en-US/DEBS/ hit ENTER

sudo dpkg -i *.deb hit ENTER

Install softwares by Terminal

Open Terminal window and type:

sudo apt install python3-pip -y then hit ENTER: to install pip3 in kali.

sudo apt-get install python3-pil python3-pil-imagetk then hit ENTER: to install pillow module for python3

sudo apt-get install apache2 then hit ENTER: to install apache server on your PC.

sudo apt-get install vlc then hit ENTER: to install VLC Media Player on your PC.

vlc then hit ENTER: to lunch VLC Media Player.

Remove softwares by Terminal

Open Terminal window and type:

sudo apt-get remove -y vlc then hit ENTER: to remove vlc player

sudo apt-get --purge remove vlc then hit ENTER: to remove configration file of any software after removing that software.

pip install commands for python3

sudo pip3 install kivy then hit ENTER: to install Kivy for python3.

sudo pip3 install playsound then hit ENTER: to install playsound module for python3.
