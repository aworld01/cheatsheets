Ways to be annonymous:-

  1. Installing and Configuring TOR with Proxychains

    2. sudo apt-get install tor: to install tor.

    sudo apt-get install proxychains: to install proxychains

    locate proxychains.conf: to locate where 'proxychains.conf' file is.

    nano /etc/proxychains.conf: to edit 'proxychains.conf' file.

    The edited changes will look like this:-

        dynamic_chain
    #strict_chain
    
    socks4  127.0.0.1 9050
    socks5  127.0.0.1 9050

    service tor start: to start tor service.

    service tor stop: to stop tor service.

    service tor restart: to restart tor service.

    tor-resolve w3schools.com: to get ip address of w3schools.com

    proxychains firefox: to open 'firefox' with proxychains.

    proxychains nmap -sS -v 66.29.212.110

  2. Install Tor Browser in Kali Linux using apt in Terminal

    3. Open Terminal and fire the commands below:-

    sudo apt-get install tor torbrowser-launcher: to install TorBrowser

    sudo torbrowser-launcher: to open the Tor Browser

  3. Anonsurf tool

    4. git clone https://github.com/Und3rf10w/kali-anonsurf.git: to download 'anonsurf' tool.

    cd kali-anonsurf: to go into kali-anonsurf directory.

    sudo bash installer.sh: to install anonsurf tool.

    anonsurf start: to start anonsurf.

    anonsurf status: to check status of anonsurf.

    anonsurf myip: to check your ip.

    anonsurf restart: to restart anonsurf.

    anonsurf stop: to stop anonsurf.

  4. 4nonimizer tool

    5. git clone https://github.com/Hackplayers/4nonimizer: to download '4nonimizer' tool.

    cd 4nonimizer: to go into 4nonimizer directory.

    ./4nonimizer install: to install 4nonimizer tool.

    4nonimizer start: to start 4nonimizer.

    4anonsurf stop: to stop 4nonimizer stop.

    4nonimizer help: to get help about 4nonimizer.

  5. Nipe tool

    6. git clone https://github.com/GouveaHeitor/nipe: to download 'nipe' tool.

    cd nipe: to go into nipe directory.

    chmod +x setup.sh: to give permision.

    ./setup.sh: to install nipe tool.

    perl nipe.pl start: to start nipe.

    perl nipe.pl status: to check status of nipe.

    perl nipe.pl restart: to restart nipe.

    perl nipe.pl stop: to stop nipe.
