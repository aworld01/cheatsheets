Installation NetHunter

  1. Search: [termux] #https://f-droid.org/en/packages/com.termux/

    2. Download and Install

  2. termux-setup-storage: to give storage permission.
  3. pkg update: to update all URLs of pkg package manager.
  4. pkg upgrade -y: to upgrade pkg package manager.
  5. apt update: to update all URLs of apt package manager.
  6. apt upgrade -y: to upgrade apt package manager.
  7. pkg install wget: to install webget package.
  8. wget -O "script-name" https://offs.ec/2MceZWr: to download nethunter script
  9. ls: to check the downloaded script.
  10. chmod +x "scrip-name": to give executabale permission.
  11. ./"scrip-name": to install nethunter

Usage:

Open Termux and type one of the following::-

  1. nh -r apt update: to run update command in NetHunter environment as root
  2. nh pwd: to run pwd command in NetHunter environment
  3. nh: to start Kali-NetHunter command line interface
  4. nh -r: to start Kali-NetHunter CLI as root
  1. Search: [NetHunter kex] #https://store.nethunter.com/packages/com.offsec.nethunter.kex/

    2. Download and Install

Usage:

Open Termux and type one of the following::-

  1. nh kex passwd: to configure the KeX password (only needed before 1st use)
  2. nh -r kex passwd: to configure the KeX password for root
  3. nh kex &: to start Kali NetHunter Desktop Experience user sessions
  4. nh kex stop: to stop Kali NetHunter Desktop Experience
  5. nh -r kex &: to start Kali NetHunter Desktop Experience as root
  6. nh -r kex stop: to stop Kali NetHunter Desktop Experience root sessions
  7. nh -r kex kill: to kill all KeX sessions
