python3_file_IO

r = read mode (default): read a file or create a new file, if not exist. w = write mode: erase all old data and write new data or create a new file, if not exist. a = append mode: write new data next to old data or create a new file, if not exist. r+ = read + write mode: read and overwrite the file but not create a new one. (it will overwrite the file data from starting point.)

    opening_a_file

  1. f = open("Files/file.txt", mode="r", encoding="utf-8") print(f)

    2. read_to_a_file

  2. f = open("File.txt") #read mode (default) data = f.read() #to read a file. print(data) #to print data f.close() #to close the file
  3. f = open("File.txt", mode="r", encoding="utf-8") data = f.read() print(data) f.close()
  4. f = open("File.txt", mode="r", encoding="utf-8") data = f.read(5) #to read only five letters of file. print(data) f.close()
  5. f = open("File.txt", mode="r", encoding="utf-8") data = f.read(5) print(data) data2 = f.read(7) #to read next seven letters of file. print(data2) f.close()
  6. f = open("File.txt", mode="r", encoding="utf-8") data = f.readline() #to read a single line at a time print(data) data2 = f.readline() print(data2) f.close()
  7. f = open("File.txt", mode="r", encoding="utf-8") data = f.readline() print(data, end="") #to remove extra gaps between lines. data2 = f.readline() print(data2) f.close()
  8. f = open("File.txt", mode="r", encoding="utf-8") data = f.readlines() #to read all lines at a time and returns values in list. print(data) f.close()
  9. #to know the length of lines. f = open("File.txt", mode="r", encoding="utf-8") data = f.readlines() print(data) l = len(data) #to get lines of data print(f"The number of lines is: {l}") f.close()
  10. f = open("File.txt", mode="r", encoding="utf-8") data = f.readlines() for line in data: print(line, end="") f.close()
  11. f = open("File.txt", mode="r", encoding="utf-8") for line in f: print(line, end="") f.close()

    12. read_to_a_file_in_binary_mode

  12. f = open("file.txt", mode="rb") # read_binary mode data = f.read() print(data) f.close()
