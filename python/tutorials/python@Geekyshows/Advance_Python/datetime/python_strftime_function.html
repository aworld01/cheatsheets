Python strftime() function

The strftime() function is used to convert date and time objects to their string representation. It takes one or more input of formatted code and returns the string representation.

Directive Meaning Output Format
%a Abbreviated weekday name. Sun, Mon
%A Full weekday name Sunday, Monday
%w Weekday as a decimal number. 0, 1
%d Day of the month as a zero added decimal 01, 02
%-d Day of the month as a decimal number 1, 2
%b Abbreviated month name. Jan, Feb
%B Full month name. January, February
%m Month as a zero added decimal number. 01, 02
%-m Month as a decimal number. 1, 2
%y Year without century as a zero added decimal number. 00, 01
%-y Year without century as a decimal number. 0, 1
%Y Year with century as a decimal number. 2002, 2022
%H Hour (24-hour clock) as a zero added decimal number. 00, 01
%-H Hour (24-hour clock) as a decimal number. 0, 1
%I Hour (12-hour clock) as a zero added decimal number. 01, 02
%-I Hour (12-hour clock) as a decimal number 1, 2
%p Locale's AM or PM AM, PM
%M Minute as a zero added decimal number. 00, 01
%-M Minute as a decimal number. 0, 1
%S Second as a zero added decimal number. 00, 01
%-S Second as a decimal number 0, 1
%z UTC offset in the form +HHMM or -HHMM.
%Z Time zone name
