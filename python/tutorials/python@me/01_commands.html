Python3 commands

  1. python --version: to check python version.
  2. pip --version: to check python version.
  3. python.exe -m pip install --upgrade pip: to update pip
  4. pip install pytube: to install package
  5. pip uninstall pytube -y: to remove package
  6. pip pytube --version: to check version
  7. pip install pytube==15.0.0: to install a specific version of any package.
  8. python (python3 if you are using Linux OS): to lunch python interpreter.
  9. exit(): to exit from python interpreter.
  10. python hello.py (python3 hello.py if you are using Linux OS): to execute python file.
  11. pip list: to check installed python packages and version.
  12. pip help: to help.
