greenColor for 'editMode' blueColor for 'CommandMode'


  1. Esc to 'CommandMode'
  2. Enter for 'EditMode'
  3. Esc, m: to markupMode
  4. Esc, y: to code
  5. CMD, EuroUp and EuroDown to navigate between cell.
  6. CMD, Spacebar to scroll down.
  7. CMD, Shift+Spacebar to scroll up.
  8. CMD, a to add new cell above the current cell.
  9. CMD, b to add new cell below the current cell.
  10. CMD, Shift+downEuro to select below cell.
  11. CMD, Shift+upEuro to select above cell.
  12. CMD, select,x to cut cell.
  13. CMD, select,v to paste cell.
  14. CMD, select,c,v to copy cell.
  15. CMD, select,D*2 to delete cell.
  16. Ctrl+Enter to run cell.
  17. Shift+Enter to run cell and add new cell below.
  18. CMD, z to undo cell.
  19. !python --version: to check python version (running cmd command)
  20. torch.__version__: to check any installed packages version
